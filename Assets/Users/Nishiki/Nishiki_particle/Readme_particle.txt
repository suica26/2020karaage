エフェクト一覧

　ｴﾌｪｸﾄ自体は全部で 9 種類。


➡bonusbrast

　進化したときのエフェクト。
　形態が変わった直後に発生する。


➡coinsplash

　お金のエフェクト。
　敵を倒した時に発生する。

　script の itemexplosion は
　爆発させる役割を持つ。
　大量の子オブジェクトを
　乱雑に破裂させているﾌﾟﾛｸﾞﾗﾑ。

　他オブジェクトに
　itemer という script を入れて
　このｴﾌｪｸﾄを
　呼び出す形式を取っている。


➡cutterfire

　その名の通りカッターの火花。
　落下カッター攻撃　つまり
　地面に突き刺した時に発生する。


➡damageeffect

　被弾のエフェクト。
　自分がダメージを
　受けた際に発生する。
　

➡ene icon

　敵のアイコン。
　見た目は　2D だけど
  カメラの角度に合わせて追尾する。
　でもダメ出し喰らったから
　没になる可能性 大。
　実装してみるだけ
　してみてほしい感（？）もある。
　

➡enedeatheffect

　コインドロップの煙エフェクト。
　正直無くても良さそう。

　他オブジェクトに
　itemer という script を入れて
　このｴﾌｪｸﾄを
　呼び出す形式を取っている。
　コインとセット。２つで１つ。
　ﾊｯﾋﾟｰｾｯﾄ


➡explosion2d

　キックの爆発エフェクト。
　見た目は　2D だけど
  カメラの角度に合わせて追尾する。
　蹴りが命中したときに発生する。


➡fall attack smoke

　落下攻撃エフェクト。
　技「落下攻撃」をしたときに発生する。
　強烈な一撃に見える。
　これ本命。必ず入れてあげてね。


➡morning brast

　新規おはブラエフェクト。
　技「おはようブラスト」をしたときに
　発生する。
　途中で出現する水色のサークルは
　必要に応じて消してあげてください。
　（見栄えの問題）


➡reality explode

　現実味の爆発エフェクト。
　ビル・家屋倒壊、自動車爆発　etc
　何かを破壊したときに発生する。
　ただしガソリンスタンドの
　破壊とは別。
　リアルすぎるので没になるかも。
